# Roavooo 🌍

Roavooo is a travel discovery and trip planning web app that helps users explore cities, find places (stays, activities, restaurants), and organize their trips in one platform.

## 🚀 Live Demo
https://roavooo.vercel.app

## 🛠 Tech Stack
- React
- JavaScript
- Supabase (Database & Auth)
- Tailwind CSS

## ✨ Features
- Browse cities and places
- Categories: stays, activities, restaurants
- Search with filters
- Favorites system (in progress)
- Multi-language support (EN / FR)

## 📌 Status
MVP in progress — actively improving and adding features.
